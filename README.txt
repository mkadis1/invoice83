============================================
  Invoice83 - Navodila za namestitev
============================================

ZAHTEVE
-------
- Windows 10 ali novejši
- Python 3.11 ali novejši
  Prenos: https://www.python.org/downloads/
  POZOR: Pri namestitvi Pythona obvezno obkljukajte
         "Add Python to PATH"!

- (Neobvezno) Tesseract OCR - samo za skeniranje PDF računov
  Prenos: https://github.com/UB-Mannheim/tesseract/wiki
  Namestite v privzeto mapo: C:\Program Files\Tesseract-OCR\


NAMESTITEV (samo prvič)
-----------------------
1. Razpakirajte ZIP datoteko v poljubno mapo
   (npr. C:\Invoice83\)

2. Dvokliknite datoteko: NAMESTITEV.bat

3. Počakajte, da se namestitev zaključi
   (lahko traja 2-5 minut - prenaša Python pakete)

4. Po uspešni namestitvi dvokliknite: ZAGON.bat


ZAGON (vsakič)
--------------
Dvokliknite: ZAGON.bat

Program se odpre v vašem spletnem brskalniku
na naslovu: http://127.0.0.1:8000

POZOR: Okno ukazne vrstice (cmd) mora ostati
odprto med uporabo programa. Zapiranje okna
ustavi program.


PRVA UPORABA
------------
1. Pojdite v "Nastavitve" (ikona zobnika)
2. Vnesite podatke vašega podjetja
3. Začnite z delom!
